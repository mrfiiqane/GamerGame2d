# 🎮 Jumper Game (2D Platformer)

A simple 2D platformer game built with **Godot Engine**. This is a beginner-friendly project focused on smooth movement, jumping mechanics, and basic level design.

---

## 🚀 Features

- Smooth 2D player movement  
- Jumping & platform physics  
- Simple level design  
- Lightweight and fast  
- Windows PC support

## 💻 How to Run Source (Godot required)

1. Download or clone this project
2. Open **Godot Engine**
3. Click **Import Project**
4. Select `project.godot`
5. Press **Play (F5)**

---

## 🎮 Controls

- Move Left: `A` / `←`
- Move Right: `D` / `→`
- Jump: `Space`

---
