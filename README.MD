\# 🎮 2D Platformer Game



A simple 2D platformer game built with \*\*Godot Engine\*\*. This is a beginner-friendly project focused on smooth movement, jumping mechanics, and basic level design.



\---



\## 🚀 Features



\- Smooth 2D player movement  

\- Jumping \& platform physics  

\- Simple level design  

\- Lightweight and fast  

\- Windows PC support  



\---



\## 💻 How to Play (PC)



\### ▶️ Option 1: Run Source (Godot required)

1\. Download or clone this project

2\. Open \*\*Godot Engine\*\*

3\. Click \*\*Import Project\*\*

4\. Select `project.godot`

5\. Press \*\*Play (F5)\*\*



\---



\### 📥 Option 2: Download PC Game (No Godot needed)



1\. Go to \*\*GitHub Releases\*\*

2\. Download the latest version:

&#x20;  - `2D-Platformer-Windows.zip`

3\. Extract the file

4\. Run `game.exe`



👉 Download here:  

https://github.com/USERNAME/REPO/releases

\## 🎮 Controls



\- Move Left: `A` / `←`

\- Move Right: `D` / `→`

\- Jump: `Space`
